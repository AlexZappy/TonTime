{\bfseries{Ton\+Time}} is an Arduino library that replicates the TON (on-\/delay) timer logic found in industrial PLCs. It enables you to trigger an output only after an input has remained active for a fixed duration, using non-\/blocking {\ttfamily millis()} logic

\DoxyHorRuler{0}
\hypertarget{md__r_e_a_d_m_e_autotoc_md13}{}\doxysection{\texorpdfstring{‚öôÔ∏è Features}{‚öôÔ∏è Features}}\label{md__r_e_a_d_m_e_autotoc_md13}

\begin{DoxyItemize}
\item PLC-\/style TON (on-\/delay) timer logic
\item Fully non-\/blocking (uses {\ttfamily millis()})
\item Delay time set at construction
\item Output Q becomes {\ttfamily true} only after the preset time has elapsed
\item Output remains {\ttfamily true} while the input stays high
\item Resets immediately when the input drops (unless latch mode is implemented)
\item Provides utility functions to track elapsed and remaining time
\end{DoxyItemize}
